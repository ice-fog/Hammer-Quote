Загрузите распакованный архив в бинарном (двоичном) режиме
Создайте новую базу MySQL на своем сервере.
Импортируйте файл db.sql в созданную базу.
Отредактируйте файл "/config/db-config.php" изменить данные в кавычках '' на свои данные.
Установите опцию CHMOD 777 для папок:
data
files
files/images
files/images/thumbnail

Для входа в административную панель перейдите по адресу: имя сайта/admin
Логин admin
Пароль admin

Откройте раздел "Настройки" и отредактируйте свой профиль.
